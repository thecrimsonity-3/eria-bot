Eria Bot

Privacy Policy

Eria Bot may collect and store limited data necessary for its functionality.

Data We Collect

  • Discord User IDs
  • Discord Server (Guild) IDs
  • Channel IDs (for tickets and moderation systems)
  • Message IDs (for logging purposes)
  • Server configuration settings

We do NOT collect sensitive personal information such as passwords, emails, or payment data.

How We Use Data

Data is used only for bot functionality such as moderation, ticket systems, and configuration management.

Data Sharing

We do not sell, trade, or share your data with any third parties.

Data Removal

Server owners may request data deletion by removing the bot from their server.
